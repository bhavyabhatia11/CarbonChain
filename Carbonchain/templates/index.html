CarbonCredit Exchange for Apparel Industry

This is a prototype of a carbon credit network built over blockchain implemented using python and flask. It aims to incentivise Recycling of clothing products by a reward point system. This prototype was buit for the partial fulfillment of the course MF-412 Sustainable Manufacturing. For more details read the project report.
